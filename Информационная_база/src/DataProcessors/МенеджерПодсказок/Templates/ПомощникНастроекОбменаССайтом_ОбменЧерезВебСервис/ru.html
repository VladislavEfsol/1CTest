Для определения параметров подключения к Web-сервису необходимо выполнить его публикацию. Для этого в Конфигураторе информационной базы в окне Публикация на веб-сервере (меню Администрирование/Публикация на веб-сервере...) нужно задать параметры публикации и установить флажки использования Web-сервиса.

После этого, в браузере можно проверить возможность подключения к Web-сервису.

Проверить работу веб-сервиса в 1С:Fresh можно, вбив в строку браузера: https://1cfresh.com/a/sbm/<номер области на фреше>/ws/SiteExchange?wsdl

При корректной работе веб-сервиса в браузере появится окно авторизации, и затем откроется схема описания веб-сервиса.

Прочитать про подключение к веб-сервису можно на ИТС: https://its.1c.ru/db/metod81#content:4935:hdoc

В 1С:УНФ есть два веб-сервиса для обмена с сайтом:

SiteExchange - обмен в формате CommerceML 2.05

Методы:

GetAmountAndPrices - цены и количество

Параметры:

  • ModificationDate - Тип: Дата в формате dateTime (http://www.w3.org/2001/XMLSchema)

Пример: ДатаФормирования="2018-07-26T17:51:30"


  • GroupCode - Тип: Строка, код группы Номенклатуры, по которой будет установлен отбор на возвращаемые данные.

Пример: КодГруппы="ФР-00000706"


  • WarehouseCode - Тип: Строка, код склада, по которой будет установлен отбор на возвращаемые данные.

Пример: Склад="ФР-000001"


  • OrganizationCode - Тип: Строка, код организации, по которой будет установлен отбор на возвращаемые данные.

Пример: Организация="ФР-000001"


Возвращает пакет данных в формате CommerceML 2.05 (файл xml)

* Товары без установленных цен не выгружаются на сайт.


GetItems - товары

Параметры:

  • ModificationDate - Тип: Дата в формате dateTime (http://www.w3.org/2001/XMLSchema)

Пример: ДатаФормирования="2018-07-26T17:51:30"


  • GroupCode - Тип: Строка, код группы Номенклатуры, по которой будет установлен отбор на возвращаемые данные.

Пример: КодГруппы="ФР-00000706"


Возвращает пакет данных в формате CommerceML 2.05 (файл xml)


GetPicture - основная картинка товара

Параметры:

  • ItemID - Тип: Строка, уникальный идентификатор 1С

Пример: Номер1С=”fbbf5d96-6b22-11e8-877b-708bcd551b2d”


Возвращает картинку в формате base64Binary


LoadOrders - получить заказы с сайта для загрузки в 1С

Параметры:

  • OrdersData - Тип: пакет данных в формате CommerceML 2.05 (файл xml)


Возвращает булево значение - данные успешно загружены (Да/Нет)


Для всех параметров методов допускается указывать пустое значение.


SiteExchange2 (пока в тестовом режиме) - обмен в формате последний актуальный формат, сейчас поддерживается в бета-версии commerceml 2.10.

Методы:

GetAmountAndPrices - цены и количество

Параметры:

  • ModificationDate - Тип: Дата в формате dateTime (http://www.w3.org/2001/XMLSchema)

Пример: ДатаФормирования="2018-07-26T17:51:30"


  • GroupCode - Тип: Строка, код группы Номенклатуры, по которой будет установлен отбор на возвращаемые данные.

Пример: КодГруппы="ФР-00000706"


  • WarehouseCode - Тип: Строка, код склада, по которой будет установлен отбор на возвращаемые данные.

Пример: Склад="ФР-000001"


  • OrganizationCode - Тип: Строка, код организации, по которой будет установлен отбор на возвращаемые данные.

Пример: Организация="ФР-000001"


  • SettingsExchangeCode - Тип: Строка, код узла обмена, в котором указаны настройки обмена.

Пример: Код="ФР-000003"


Возвращает пакет данных в формате CommerceML 2.10 (файл xml)

* Товары без установленных цен не выгружаются на сайт.


GetItems - товары

Параметры:

  • ModificationDate - Тип: Дата в формате dateTime (http://www.w3.org/2001/XMLSchema)

Пример: ДатаФормирования="2018-07-26T17:51:30"


  • GroupCode - Тип: Строка, код группы Номенклатуры, по которой будет установлен отбор на возвращаемые данные.

Пример: КодГруппы="ФР-00000706"


  • SettingsExchangeCode - Тип: Строка, код узла обмена, в котором указаны настройки обмена.

Пример: Код="ФР-000003"

Возвращает пакет данных в формате CommerceML 2.10 (файл xml)


GetPicture - основная картинка товара

Параметры:

  • ItemID - Тип: Строка, уникальный идентификатор 1С

Пример: Номер1С=”fbbf5d96-6b22-11e8-877b-708bcd551b2d”


  • SettingsExchangeCode - Тип: Строка, код узла обмена, в котором указаны настройки обмена.

Пример: Код="ФР-000003"


Возвращает картинку в формате base64Binary


LoadOrders - получить заказы с сайта для загрузки в 1С

Параметры:

  • OrdersData - Тип: пакет данных в формате CommerceML 2.10 (файл xml)


  • SettingsExchangeCode - Тип: Строка, код узла обмена, в котором указаны настройки обмена.

Пример: Код="ФР-000003"


Возвращает таблицу результатов загрузки в формате JSON, Строка.



Для всех параметров всех методов допускается указывать пустое значение.

Если не заполнен или некорректно заполнен параметр SettingsExchangeCode, обмен выполняется в формате CommerceML 2.05 без учета настроек узла обмена.


Пример в коде 1С для SiteExchange:

Server = "https://1cfresh.com:443/a/sbm/" + НомерОбластиФреша + "/ws/SiteExchange?wsdl";

User = ПользовательВнешний;

Pass = ПарольВнешний;

Определение = Новый WSОпределения(Server, User, Pass);

Прокси = Новый WSПрокси(Определение, "http://www.1c.ru/ExchangeWithSites", "SiteExchange", "SiteExchangeSoap");

Прокси.Пользователь = User;

Прокси.Пароль = Pass;


Остатки = Прокси.GetAmountAndPrices(Дата('00010101'),,,);

Товары = Прокси.GetItems(Дата('00010101'),);

Заказы = Прокси.GetOrders(Дата('00010101'));

Картинки = Прокси.GetPicture(Строка(Новый УникальныйИдентификатор));

ПолучитьЗаказы = Прокси.LoadOrders(ПолучитьЗаказыXDTO(ТекстПакетаОбмена()));


Пример в коде 1С для SiteExchange2:

Server = "https://1cfresh.com:443/a/sbm/" + НомерОбластиФреша + "/ws/SiteExchange2?wsdl";

User = ПользовательВнешний;

Pass = ПарольВнешний;

Прокси.Пользователь = User;

Прокси.Пароль = Pass;


КодУзлаОбмена = "000000002";

Остатки = Прокси.GetAmountAndPrices(Дата('00010101'),,,,КодУзлаОбмена);

Товары = Прокси.GetItems(Дата('00010101'),,КодУзлаОбмена);

Заказы = Прокси.GetOrders(Дата('00010101'), КодУзлаОбмена);

Картинки = Прокси.GetPicture(Строка(Новый УникальныйИдентификатор), "");

ПолучитьЗаказы = Прокси.LoadOrders(ПолучитьЗаказыXDTO(Заготовка3()), КодУзлаОбмена);
